QUICK START
───────────
1.  python setup.py                        # create dirs + initialise DB
2.  cd backend
3.  pip install -r requirements.txt
4.  python app.py                          # starts on http://localhost:5000
    → Admin password printed ONCE on first run – change it via profile.html

ML TRAINING (optional)
───────────────────────
1.  python data_preprocessing.py --celebdf-dir ../Celeb-DF-v1 --output-dir ../processed_dataset
2.  python train_model.py --epochs 50 --lr 0.001
3.  python evaluate_model.py

ENVIRONMENT VARIABLES (.env)
─────────────────────────────
SECRET_KEY=<generate with: python -c "import secrets; print(secrets.token_hex(32))">
JWT_SECRET_KEY=<generate with: python -c "import secrets; print(secrets.token_hex(32))">
ADMIN_EMAIL=admin@yourdomain.com           # optional, defaults to admin@deepfake.com
ADMIN_PASSWORD=<strong password>           # optional, auto-generated if not set
